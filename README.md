# 📊 Sales Data Analysis

A complete Sales Data Analysis project that demonstrates how raw sales data can be transformed into meaningful business insights using Excel. This project includes data cleaning, analysis, KPI creation, Pivot Tables, Charts, and an interactive Dashboard.

---

## 📌 Project Overview

The objective of this project is to analyze sales performance and provide actionable insights that help businesses make data-driven decisions.

This project covers:

- Data Cleaning
- Data Preparation
- KPI Calculation
- Sales Trend Analysis
- Customer Analysis
- Product Performance Analysis
- Interactive Excel Dashboard

---

## 🎯 Objectives

- Analyze overall sales performance.
- Identify top-performing products.
- Discover monthly sales trends.
- Evaluate customer purchasing behavior.
- Measure profit and revenue.
- Create an interactive dashboard for business reporting.

---

## 🛠️ Tools Used
